# ИС учета и обработки заявок на ИТ-поддержку государственного учреждения

Учебное desktop-приложение на Java Swing для учета и обработки заявок IT-поддержки.

Приложение сделано как MVP информационной системы ServiceDesk для государственного учреждения.

## Технологии

- Java 17
- Java Swing
- SQLite
- JDBC
- Maven
- jpackage для сборки desktop-запускателя

## Возможности

### Авторизация

В системе есть авторизация по логину и паролю.

Тестовые пользователи:

| Логин | Пароль | Роль |
|---|---|---|
| `user` | `1234` | Сотрудник |
| `it` | `1234` | IT-специалист |
| `admin` | `1234` | Администратор |

## Роли

### Сотрудник

Сотрудник может:

- создавать заявку;
- просматривать свои заявки;
- добавлять комментарии;
- видеть статус заявки;
- видеть исполнителя;
- видеть отчет IT-специалиста о выполнении заявки.

### IT-специалист

IT-специалист может:

- просматривать список заявок;
- брать заявку в работу;
- менять статус заявки;
- добавлять комментарии;
- писать отчет о выполнении;
- отмечать заявку выполненной;
- закрывать заявку.

### Администратор

Администратор может:

- просматривать все заявки;
- назначать заявку конкретному IT-специалисту;
- менять приоритет и срок исполнения;
- управлять пользователями;
- назначать роли пользователям;
- просматривать статистику по заявкам.

## Статусы заявок

- `NEW` — новая
- `IN_PROGRESS` — в работе
- `DONE` — выполнена
- `CLOSED` — закрыта

## Приоритеты заявок

- `LOW` — низкий
- `NORMAL` — обычный
- `HIGH` — высокий
- `URGENT` — срочный

## База данных

Используется SQLite.

При первом запуске приложение автоматически создает:

- таблицы;
- тестовых пользователей;
- базовые категории заявок.
