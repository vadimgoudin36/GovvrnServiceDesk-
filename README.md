# ServiceDesk Министерства региональной безопасности Воронежской области

Учебное desktop-приложение на Java Swing для учета и обработки заявок IT-поддержки государственного учреждения.

## Возможности

- авторизация пользователей по логину и паролю;
- роли `EMPLOYEE`, `SPECIALIST`, `ADMIN`;
- создание и просмотр заявок сотрудником;
- комментарии к заявкам;
- назначение заявки администратором IT-специалисту;
- взятие заявки в работу IT-специалистом;
- смена статуса заявки;
- отчет IT-специалиста о выполнении;
- управление пользователями и ролями;
- статистика по заявкам;
- журнал действий с фиксацией автоматических и ручных изменений.

## FR-05: автоматическое изменение статуса

В проекте реализованы все 4 правила из требования:

1. `SLA`: если заявка находится в статусе `NEW` больше 2 часов, статус автоматически меняется на `OVERDUE`.
2. `Срочная заявка`: если в описании есть слова `срочно`, `не работает` или `критично`, статус меняется на `URGENT_REVIEW`.
3. `Эскалация`: если по заявке оставлено 3 или больше комментария, а статус ни разу не менялся, статус меняется на `ESCALATION`.
4. `Автозакрытие`: если IT-специалист пишет комментарий со словом `исправлено` или `готово`, статус меняется на `DONE`. Если затем автор заявки пишет `спасибо`, статус меняется на `CLOSED`.

Все автоматические изменения записываются во вкладку администратора `Журнал действий`.

## Запуск через Maven

Откройте `cmd` в папке проекта и выполните:

```cmd
"C:\Program Files\Apache\apache-maven-3.9.16\bin\mvn.cmd" clean compile exec:java
```

## Сборка JAR

```cmd
"C:\Program Files\Apache\apache-maven-3.9.16\bin\mvn.cmd" clean package
```

После сборки запускаемый файл будет здесь:

```text
target\ServiceDesk-MRB-Voronezh-Desktop.jar
```

Запуск JAR:

```cmd
java -jar target\ServiceDesk-MRB-Voronezh-Desktop.jar
```

## База данных

При первом запуске приложение автоматически создает таблицы и тестовых пользователей.

Файл SQLite создается в рабочей папке проекта:

```text
servicedesk.db
```

## Статусы заявок

- `NEW` - новая;
- `IN_PROGRESS` - в работе;
- `DONE` - выполнена;
- `CLOSED` - закрыта;
- `OVERDUE` - просрочена;
- `URGENT_REVIEW` - требует срочного рассмотрения;
- `ESCALATION` - требует эскалации.

HTML-документация классов, методов и пакетов проекта находится в каталоге
[`docs/javadoc`](docs/javadoc/index.html).

Чтобы пересоздать документацию, выполните из корня проекта:

```cmd
"C:\Program Files\Apache\apache-maven-3.9.16\bin\mvn.cmd" -DskipTests javadoc:javadoc
```

Maven сохранит результат в `target\reports\apidocs`.
